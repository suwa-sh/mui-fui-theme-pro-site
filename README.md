This repository contains the built assets for the MUI FUI Theme Pro landing page and Storybook documentation.

**This is NOT an open-source project.**

## About MUI FUI Theme Pro

MUI FUI Theme Pro is a premium Futuristic UI (FUI) theme for Material-UI (MUI).
It provides advanced components for building sci-fi inspired dashboards and applications.
